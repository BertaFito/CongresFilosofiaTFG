# Congrés de Filosofia
Pàgina web per promocionar el Congrés de Filosofia per presentar els Treballs de Fi de Grau d'un grup d'alumnes de la Facultat de Filosofia de la Facultat de Barcelona (Quadrimestre de Primavera, 2025-2026).

El contingut d'aquest repositori pertany als seus respectius creadors. No s'admet el seu ús sense el consentiment explícit dels creadors del contingut.
